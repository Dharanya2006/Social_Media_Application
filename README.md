# Social Media Application (Java Console Project)

## 📌 Project Description

The **Social Media Application** is a console-based Java program that simulates basic social networking features. Users can register, log in, send friend requests, create posts, view feeds, and receive notifications.

This project demonstrates **Object-Oriented Programming concepts in Java** such as classes, objects, collections, and menu-driven applications.

---

## 🚀 Features

* User Registration
* User Login
* View Friends
* Send Friend Requests
* View Friend Requests
* Create Posts
* View Feed
* Like and Comment on Posts
* View Notifications
* Logout

---

## 🛠️ Technologies Used

* Java
* OOP Concepts
* Java Collections
* Console-based Interface
